# Quantum Algorithm Simulation using Qiskit

This project explores fundamental quantum algorithms through simulation using Qiskit.

## Overview

- Quantum Fourier Transform (QFT)
- Inverse QFT (IQFT)
- Quantum Phase Estimation (QPE)
- Entanglement (Bell and GHZ states)

## Key Features

- Implementation of scalable quantum circuits
- Analysis of phase estimation accuracy
- Log(error) vs qubit scaling
- Demonstration of exponential error decay

## Results

- Error decreases exponentially with number of qubits
- Binary representation of phase strongly affects convergence
- Exact binary phases converge faster than non-terminating ones

## Tools Used

- Python
- Qiskit
- NumPy
- Matplotlib
